◆ DiscardThreshold

float DarkRift.Server.Plugins.Matchmaking.RankingMatchmaker< T >.DiscardThreshold
get

The threshold at which a match will be immediately discarded.

When the matchmaker analyzes the set of entities to find possible matches it calculates the suitability metric for all entity pairs. If the score returned by GetSuitabilityMetric(T, T, MatchRankingContext<T>) is below the DiscardThreshold it will not be considered any further in trimming the search tree substantially.

Increasing this value will improve performance and memory usage but may return worse results and cause longer wait times.

◆ GroupDiscardThreshold

float DarkRift.Server.Plugins.Matchmaking.RankingMatchmaker< T >.GroupDiscardThreshold
get

The threshold at which a match will be immediately discarded.

When the matchmaker analyzes the set of entities to find possible matches it calculates the average suitability metric between each group (or group of one for individuals). If the average score calculated is below the GroupDiscardThreshold it will not be considered any further in computation therefore trimming the search tree substantially.

Increasing this value will improve performance and memory usage but may return worse results and cause longer wait times.
